# Decision Harness Host

This workspace is a dedicated `bagakit-decision-harness` host.

Primary decision materials live at the host root:

- `inbox/`
- `signals/`
- `decisions/`
- `reviews/`
- `patterns/`
- `drills/`
- `ai-updates/`
- `metrics/`
- `principles/`
- `projects/`
- `exports/`

Runtime indexes and bookkeeping live under:

- `.bagakit/decision-harness/`

Do not treat `.bagakit/decision-harness/` as the source of truth for primary
decision records.
