## Current View

The strongest canonical shape for `bagakit-brainstorm` is not "simulate a
conference every time."

It is:

- a bounded planning skill with explicit artifacts
- method selection only when the problem justifies it
- clear handoff and archive destinations
- lighter defaults, with deeper expert-forum behavior as an escalation path
