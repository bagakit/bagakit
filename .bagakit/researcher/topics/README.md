# Research Topics

This directory contains canonical `bagakit-researcher` topic workspaces for the
repository.

Topic layout:

```text
.bagakit/researcher/topics/<topic-class>/<topic>/
├── originals/
├── summaries/
└── index.md
```

Topic classes should describe the working context, for example:

- `frontier`
- `research`

The topic workspace is the evidence-production surface.
It is not durable promotion by itself.
