# Design Runtime Surface

`.bagakit/design/` is the project-local runtime surface for
`bagakit-design-core`.

Use this root for:

- local design synthesis that helps Bagakit design work in this repository
- task-level `design-packet.toml` files
- draft, concrete-plan, and result review projections
- source-evidence, tone-axis, and rule-coverage notes that are tied to a
  concrete task

Do not use this root as a second `docs/` tree. Stable public rules belong in
`docs/` or in the owning skill references. Task-level packet state belongs in
`design-packet.toml`; Markdown files here should be summaries, reviews, or
evidence notes.

Current local synthesis files:

- `brand-tonality-synthesis.md`
- `design-rule-synthesis.md`
- `review-protocol.md`

Stable contract:

- `docs/specs/runtime-surface-contract.md`
