# Bagakit Codex Webpage Design Eval

Non-gating evals for `bagakit-codex-webpage-design`.

The first suite is a historical failure bench. It does not judge a rendered
page. It checks whether the skill text, references, and bench data preserve the
failure patterns that should block future webpage-design runs.

Historical failures come from selector/evolver/Spark task evidence and should
graduate only when a future browser-backed case can reproduce and score the
same failure mode.
