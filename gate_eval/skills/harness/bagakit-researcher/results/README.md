# Researcher Eval Results

Generated run artifacts belong under:

- `gate_eval/skills/harness/bagakit-researcher/results/runs/<run-id>/`

Expected contents for each run:

- `summary.json`
  - aggregate case status and focus coverage
- `cases/<case-id>.json`
  - per-case assertions, commands, and normalized outputs

This directory is intentionally non-gating.
