# Pre-Draft Route Memo Template

这份 memo 只解决一个漂移问题：正文还没开写，标题承诺、首问和证据顺序已经在来回滑动。

## 什么时候用

在主 lane 已经选定、准备起草长文之前用。它不是全文提纲，只负责把选卡前的 route 状态压成一个可执行起点。

## 填写顺序

1. 先写 `title_promise`，不要先想漂亮标题。
2. 再写 `first_question`，确认正文第一步在回答什么。
3. 然后写 `evidence_movement` 和 `chapter_movement`，把结构推进钉住。
4. 最后写 `exit_move`，确认读者读完后真正带走什么。

## 模板

```md
# Pre-Draft Route Memo

- title_promise:
- first_question:
- evidence_movement:
- chapter_movement:
- exit_move:
```
