## Hard Gate

- [ ] 开头 15% 内有明确问题和核心命题

- [ ] 至少一个核心 H2 有例子、证据、机制解释或反例
- [ ] 内部黑话/缩写首次出现时已定义
- [ ] 有真实场景、失败案例或具体应用场景
- [ ] 结尾回到主命题
- [ ] 首屏 3 段内兑现标题承诺
- [ ] 没有无依据的人群泛化或拉踩表达
- [ ] 无上下文读者能在首屏理解文章主张和 stakes
