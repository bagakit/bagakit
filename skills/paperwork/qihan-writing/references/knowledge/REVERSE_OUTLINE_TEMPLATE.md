# Reverse Outline Template

这份模板只解决一个漂移问题：看范文时容易抄句子、不抄结构，结果复用不到自己的正文。

## 什么时候用

当你在读代表文章、卡片范文或外部 exemplar，想把它压成“结构动作地图”而不是摘抄集时使用。

## 填写规则

1. 只记段落职责、证据动作和转场作用，不抄整段原文。
2. 单条 `source_line` 最多保留一句极短锚点，用来提醒动作，不用来攒金句。
3. 每读完一篇，最后必须写 `steal_for_our_draft` 和 `do_not_copy`。

## 模板

```md
# Reverse Outline

- source:
- link_or_handle:
- article_promise:
- why_this_exemplar:

## Structure Map
- block_1:
  - job:
  - source_line:
  - evidence_move:
  - transition_role:
- block_2:
  - job:
  - source_line:
  - evidence_move:
  - transition_role:
- block_3:
  - job:
  - source_line:
  - evidence_move:
  - transition_role:

## Reusable Moves
- opening_move:
- section_order_logic:
- paragraph_rhythm:
- evidence_pattern:

## Transfer
- steal_for_our_draft:
- must_rewrite_for_our_object:
- do_not_copy:
```
