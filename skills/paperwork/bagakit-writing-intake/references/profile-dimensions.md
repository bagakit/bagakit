# Profile Dimensions

Use these dimensions to describe observed language behavior. Record only what
the evidence supports.

## Task And Audience

- `audience_specificity`: named reader, role, expertise level, or vague public
- `channel_fit`: chat, Feishu doc, technical article, report, note, speech, slide
- `action_orientation`: inform, decide, persuade, teach, align, unblock
- `domain_load`: how much domain context the prose expects the reader to have

## Structure

- `opening_move`: conclusion-first, question-first, scene-first, definition-first
- `argument_order`: problem -> mechanism -> evidence -> action, or another order
- `section_granularity`: few broad sections, nested structure, compact memo
- `transition_style`: explicit signposts, implicit logical movement, abrupt jumps
- `closure_shape`: summary, next action, warning, open question, decision rule

## Sentence And Paragraph Shape

- `sentence_density`: compact, medium, expansive
- `paragraph_density`: one-idea paragraphs, layered paragraphs, list-heavy
- `verb_posture`: concrete action verbs, static nouns, abstract process verbs
- `modifier_load`: sparse, calibrated, overloaded
- `rhythm`: short pulses, varied cadence, long accumulative movement

## Evidence And Reasoning

- `evidence_posture`: examples, metrics, mechanisms, citations, lived cases
- `claim_boundary`: precise thresholds, cautious qualifiers, broad assertions
- `counterexample_handling`: absent, acknowledged, integrated, central
- `abstraction_ladder`: concrete-first, concept-first, mixed, too abstract
- `mechanism_visibility`: names actions and causal links, or hides them

## Voice And Tone

- `persona_distance`: first-person, team voice, neutral analyst, editorial voice
- `warmth`: plain, warm, sharp, formal, playful, severe
- `authority_posture`: tentative, grounded, directive, over-assertive
- `metaphor_use`: none, occasional, structural, decorative
- `emotional_charge`: low, restrained, motivating, urgent, alarmist

## Style Candidate Discipline

For each dimension, separate:

- observation: what the sample does
- candidate rule: what future writing might do
- scope: when the rule should apply
- rollback: when the rule should be rejected

Do not convert one attractive sample into a broad personal rule without enough
evidence.
