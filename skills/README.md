# Skills Tree

This directory is the long-term home for installable Bagakit skill sources.

The required layout is family-based:

- `skills/harness/`
- `skills/swe/`
- `skills/paperwork/`
- `skills/gamemaker/`
- `skills/human-improvement/`

Flat roots such as `skills/<skill-id>/SKILL.md` are forbidden.

The only supported installable protocol is:

- `skills/<family>/<skill-id>/SKILL.md`
