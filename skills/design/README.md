# Design Family

This family groups skills whose primary authority is visual, interaction,
brand, product, or experience-design judgment. A design skill may still ship
code when the code is the proof surface for the design.

Design skills may be composed with SWE skills, Figma workflows, image
generation, or paperwork skills, but they should keep their own design
contracts.

Representative skills in this family include:

- `bagakit-design-core`
- `bagakit-codex-webpage-design`
